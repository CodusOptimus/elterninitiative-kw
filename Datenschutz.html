Hosting (GitHub Pages) & CI (GitHub Actions)

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf können durch GitHub (GitHub Inc., USA) technisch notwendige Daten verarbeitet werden (z. B.
